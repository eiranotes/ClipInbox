# Visual target

## Goal

Keep the approved seven-frame marketing composition and real SwiftUI surfaces, but make the in-app sample library read as a varied personal archive instead of a repeated beach demo.

## Content map

1. Recipe: overhead lemon pasta, cobalt bowl, terracotta linen.
2. City walk: quiet hanok lane, timber, tile, leaf green, blue sky.
3. Studio reference: terracotta task lamp, teal notebook, oak desk.
4. Memo: text-led travel checklist with no image.
5. Exhibition reference: existing blue data/poster-style graphic.
6. Beach photo: existing beach image appears once only, below the first screenful.
7. Product writing: existing yellow lightbulb graphic.
8. Reading: text-led clipping with no image.
9. Weekly review: text-led memo with no image.
10. Interior: existing warm living-room photo.

The first visible Inbox rows must alternate subject, hue, and content type. The link-detail feature frame uses the hanok image, not the beach image.
