# Design Tokens

Source: `DESIGN.md`

## Direction

Clip Inbox keeps the warm ivory board surface and thick Trello-like ownership, but badges and tags now use a restrained state system instead of per-content pastel colors.

## Accent / State Set

- Yellow `#FFD900`: primary CTA, selected control, new/unsorted state.
- Blue `#BBD7FF`: informational content type marker and focus ring.
- Green `#9BE7B0`: saved/success state.
- Danger `#FF4B4B`: destructive confirmation only.

## Chip / Badge Rule

Tags are neutral by default with a soft token border. Badges may add a small state dot, but content type badges all share the blue informational marker.
